# 출력 품질 evals (수동)

5-Color Harness 스킬의 출력은 창의적 system instruction이라 자동 grader로 채점하기 어렵다. 같은 입력을 3회 호출해 (1) 항목별 체크리스트 통과 여부와 (2) 회차 간 일관성을 사람 손으로 본다.

`EVALS_GUIDE.md`의 합격 기준은 **9개 항목 × 5개 케이스 × 3회 = 135/135**다. 한 항목이라도 한 번이라도 빠지면 fail.

---

## 케이스 5종

| ID | 입력 | 합격선 | 검증 포인트 |
|----|------|--------|------------|
| Q1 | `이메일` | 9.5 | 외부 커뮤니케이션 카테고리. 분량 1500-2800자. 종결 체 `~합니다·~드립니다`. |
| Q2 | `사과문` | 9.7 | 9.7군 강도. 직접 인용 금지 2줄 이상("심려를 끼쳐 드린 점 유감", "We apologize if" 등). BLACK 제약 2개. |
| Q3 | `블로그` | 9.0 | 9.0군 절제. 특화 금지 1-2개. 작성자 톤 우선. 정량 강제 룰 적용 안 됨. |
| Q4 | `영업 덱이랑 발표 스크립트 같이` | 9.5 | 복합 산출물. 본체(덱)+부속(스크립트) 흡수. GOLD 단일 독자(60분 강연장 청중). |
| Q5 | `make a system prompt for board memo` | 9.7 | 영문 트리거. english-augment 자동 적용. BLACK·GOLD 영문 환경 인물. 분량 700-1700 words. 평가 발화 영문. |

## 항목별 체크리스트 (각 출력에 대해)

다음 9항목을 모두 통과해야 한 번의 합격이다. 한 케이스당 3회 호출, 모두 통과해야 그 케이스 합격이다.

- [ ] 출력이 마크다운 한 묶음이고 끝줄에 "위 마크다운을 새 클로드 프로젝트의 지침 박스에 붙여 넣어 사용하세요" 안내가 있다.
- [ ] BLACK 캐스팅에 연차(15년차+), 톤 한 가지, 제약, 분량 가이드, 종결 체가 모두 들어 있다.
- [ ] RED·SILVER·BLUE·GOLD 정체성에 머묾(RED는 이성, SILVER는 분야, BLUE는 공감, GOLD는 1인칭 반응). 침범한 발언은 무효.
- [ ] 종결 체가 가이드 표대로 한 가지로 통일됐다(PPT 헤드라인+본문 분리는 명시 시 허용).
- [ ] 본문에 em dash(`—`, `–`) 0개. 한 개라도 있으면 자동 fail.
- [ ] 추측 어미·약화어·지시대명사·메타 표현 본문 0회.
- [ ] 비평가 발화 톤 셋(RED·SILVER·BLUE)이 서로 다르다.
- [ ] GOLD가 시간·장소·행위·심리가 들어간 한 줄 장면이다(인구통계 아님).
- [ ] 합격선 강도 일치. 9.7군이면 특화 금지 4-5개 + BLACK 제약 2개 + 직접 인용 2줄. 9.0군이면 특화 금지 1-2개로 절제.

---

## 회차 간 일관성 (3회 호출의 변주 폭)

같은 입력에 대해 3회 호출했을 때 다음이 같은 범위 안에서 변주되는지 본다.

- [ ] BLACK 톤 단어가 비슷한 방향(예. "단정·드라이·간결" 풀 안)에서 흔들린다. 1회는 "차분", 2회는 "단정", 3회는 "신랄"처럼 인접 톤이면 합격. 1회는 "차분", 2회는 "활기찬", 3회는 "장난스러운"처럼 점프하면 fail.
- [ ] GOLD 시간·장소·행위가 같은 작업 영역의 표준 장면을 반복한다. 이메일이면 모두 "월요일 아침 받은 편지함 미리보기 두 줄" 계열, 사과문이면 모두 "발송 직후 수신자가 본문을 처음 읽는 순간" 계열.
- [ ] 합격선 보정이 매번 동일하다. 같은 입력인데 1회는 9.5, 2회는 9.7로 흔들리면 fail.
- [ ] 분량 범위가 동일 군 안에서 변주된다(예. 모두 9.5군 2000-2800자 안).

## 빠른 60초 sanity check

여기까지 다 돌릴 시간이 없을 때.

1. 새 클로드 프로젝트를 만들고 SKILL.md 본문을 지침 박스에 붙인다.
2. 채팅에 "이메일"이라고만 친다(Q1 1회).
3. 위 9개 항목 체크리스트가 모두 통과하는지 확인한다.
4. "사과문" 한 번 더 쳐서(Q2 1회) 9.7군 처리(직접 인용 금지 2줄 이상)가 들어가는지 확인한다.

이 둘이 통과하면 일상 사용 OK 수준(9.5/10). 진짜 10점은 위 5개 × 3회 × 9항목을 다 돌려야 한다.
