# 분석·보고서

이 카테고리는 결정자(임원·이사회·LP·딜 매니저·애널리스트)가 분석 결과를 받아 다음 행동을 정하도록 만드는 모든 분석·보고서를 다룬다. 1페이지 실무 보고서·분석 보고서·회의록·IR 발표·OKR/KPI 리포트·MD&A·LP 리포트·DD 메모·산업 분석·전문가 콜·진단 보고서·엑셀 모델·시장 분석·실사 보고서가 여기 속한다. 공통점은 (1) 통찰 한 줄이 결론보다 먼저 잡혀야 한다, (2) 숫자에 출처가 붙어야 한다, (3) 가정·전제를 명시하지 않으면 결론이 흔들린다, (4) 결론 섹션은 메모 톤(단정형 종결어미·행동 동사·주체 명시)으로 닫는다.

## 분석 보고서의 결론 원칙

이 카테고리의 산출물은 **형식상 분석 보고서지만, 요약(Executive Summary)과 결론 섹션은 메모 톤으로 써야 한다.** 전략기획(시장·경쟁사·트렌드 분석, 캠페인 성과, 데이터 분석, 사내 진단 메모), 회계·재무(MD&A, 분기 실적 리뷰, 재무 분석), M&A(실사 보고서, 비즈니스 DD)에 동일하게 적용된다.

- 형식 유지. 사실관계·쟁점·검토·결론의 정형 구조, 인용·출처·각주, 부록은 그대로 둔다.
- 메모 톤 정의. 결론 한 줄에 (a) 단정형 종결어미("~함", "~이다", "~권고"), (b) 행동 동사(진행·보류·수정·이행·거부·승인·재배분·축소·확대·정정), (c) 주체 명시(팀장·실장·본부장·임원, 또는 클라이언트·이사회·재무팀장) 셋이 동시에 들어간다. "사료된다", "보아야 할 것이다", "추가 검토가 필요", "고려할 만함", "긍정적으로 검토 가능"은 (a) 단정형 결여로 자동 위반.
- 메모 톤 예시.
  - 전략기획. "Q3 캠페인은 ROAS 2.1로 목표 미달. 4분기는 신규 채널 비중을 30%로 축소하고 기존 채널 예산을 재배분할 것을 본부장께 건의함." (단정 + 행동 "축소·재배분" + 주체 "본부장")
  - 회계·재무. "당기 매출 인식 시점은 IFRS 15에 부합하지 않으며, 결산 마감 전 회계처리 정정과 외부감사 재검토를 재무팀장께 권고함." (단정 + 행동 "정정·재검토" + 주체 "재무팀장")
- 위치별 형식. 요약(Executive Summary)은 권고 한 줄 + 근거 3개 + 잔여 리스크 한 줄. 본문 결론 단락은 압축 재서술 + 단정 권고 한 줄. 후속 조치 권고 필수.
- 이유. 의사결정자는 요약과 결론만 읽는다. 일반 부서 보고는 팀장·실장·본부장 선에서 90%가 결정되고, 임원 보고는 그 위 단계에서 결정된다. 본문은 검증·방어용 갑옷이고, 결론이 실제 의사결정을 만든다.

## 종결 체 원칙

본문은 `~이다` 체로 통일한다. 결론 섹션의 메모 톤 권고 줄은 위 정의대로(단정형 + 행동 동사 + 주체).

## 매핑 표

각 행의 RED·SILVER·BLUE는 5-Color Harness 페르소나 정체성에 해당한다. RED 이성(논리·근거·결론 타당성), SILVER 분야 전문가(베테랑이 첫눈에 짚는 디테일), BLUE 공감(결정자가 부담 없이 결정에 들어가는 신뢰 무게). 분석·보고서 카테고리는 결정자(임원·LP·딜 매니저·애널리스트·재무팀장)가 BLUE의 대상이다. 합격선이 표기되지 않으면 9.5다.

## 합격선 기준 장면

- 보고서 (1페이지). "임원이 한 장으로 결정을 내리고 추가 자료를 안 청한다."
- 분석 보고서. "독자가 통찰 한 줄을 다른 회의에 그대로 옮겨 인용한다."
- 회의록. "다음 회의 직전 5분에 결정 사항만 한 번 훑고 액션 아이템 담당자·시점을 그대로 따라간다."
- IR·실적 발표 (9.7). "애널리스트 노트에 헤드라인 한 줄과 가이던스가 함께 인용된다."
- OKR 리포트. "다음 분기 OKR을 잡는 회의가 격차 원인 한 줄에서 시작한다."
- 분기 실적 리뷰. "감독 당국이 변동 원인 한 줄을 그대로 인용 가능하다. 애널리스트가 다음 화요일에 모델을 변동 원인 한 줄대로 수정한다."
- LP 리포트. "LP가 다음 펀드 약정에 그대로 들어온다."
- 비즈니스 실사 메모. "딜 매니저가 가설 검증 표 한 장으로 다음 단계(Go·Hold·Pass)를 결정한다."
- 산업 분석. "독자가 드라이버 한 줄을 자기 의사결정에 적용한다."
- 전문가 콜 노트. "투자 매니저가 핵심 인용 박스 두 줄을 IC 메모에 그대로 옮겨 적는다."
- 경영 진단. "클라이언트 임원이 ES 한 페이지로 컨설팅 권고 진행 여부와 다음 워크샵 일정을 결정한다."
- 엑셀 모델. "다른 사람이 30분 안에 가정·계산·출력 흐름을 따라가고 셀 한 개를 바꿔도 결과가 깨지지 않는다."
- 시장 분석. "독자가 통찰 한 줄을 자기 의사결정에 그대로 옮긴다."
- 실사 보고서 (9.7). "ES 한 페이지만 보고 딜 진행 여부 1차 판단이 가능하다."

## 문체·금지 뱅크

### 보고서 (실무, 1페이지)
- 종결 체. `~이다` 체. 결론 메모 톤 한 줄.
- 문체. 액션 타이틀로 시작. So what을 끝까지 견지. 한 페이지 한 메시지.
- 분량. A4 한 장, 1500~2000자.
- 금지. "검토가 필요합니다"로 마무리. "배경부터 말씀드리면"식 도입. 결론과 데이터 사이의 한 단계 비약.

### 분석 보고서
- 종결 체. `~이다` 체. 결론 메모 톤 한 줄.
- 문체. 통찰 한 줄을 헤드에. 데이터로 뒷받침. 반대 케이스 한 줄.
- 분량. 3~10페이지, 5000~15000자.
- 금지. 데이터 나열로 끝남. 통찰이 부재한 요약. "추가 분석이 필요"로 끝맺기.

### 회의록
- 종결 체. `~이다` 체.
- 문체. 결정·할 일·보류의 세 칸. 시점·담당자 명시.
- 분량. 한 페이지, 500~1000자.
- 금지. 토의 내용을 그대로 옮기기. 결정 사항을 모호하게 적기. 담당자 부재.

### IR·실적 발표 자료. 9.7
- 종결 체. 슬라이드 헤드라인 `~함·~임`. 발화 스크립트 `~이다·~합니다` 체(IR 발표 격식). 보도자료 동반은 `~다·~이다` 체.
- 문체. 첫 슬라이드에 한 줄 헤드라인 메시지(예. "QoQ 매출 +12%, 영업이익률 18% 유지"). 분기 비교는 표 한 장. 가이던스 별도 섹션.
- 분량. IR 자료 20~30장, 발화 스크립트 분당 130~150자(30분 기준 4000~4500자). 보도자료 동반은 600~1000자.
- 금지 (9.7군).
  - "견조한" 직접 인용 금지.
  - "안정적인" 직접 인용 금지.
  - "시장 상황을 면밀히 모니터링" 직접 인용 금지(가이던스 회피).
  - 출처 없는 비교 지표.
  - 형용사 가득한 자평.

### OKR·KPI 정기 리포트
- 종결 체. `~이다` 체. 격차 원인·다음 액션은 메모 톤.
- 문체. 한 페이지에 한 OKR. 목표·실측·격차 세 칸 표. 격차 원인 한 줄, 다음 액션 한 줄.
- 분량. 분기 OKR 리포트 페이지당 한 OKR, 전체 5~10페이지. 주간 KPI는 한 페이지 표 + 한 단락 코멘트.
- 금지. 빨강·초록·노랑 색만 표시하고 원인 부재. "노력하겠습니다" 류 액션 부재. 측정 기준 변경 무고지.

### 분기 실적 리뷰 (MD&A·재무 코멘터리)
- 종결 체. `~이다` 체. 항목별 변동 원인 한 줄은 단정 메모 톤.
- 문체. 항목별 변동 원인 한 줄. 정량(YoY, QoQ) + 정성(원인) 균형. 가이던스 별도 문단.
- 분량. 분기 MD&A 3~8페이지, 연차 MD&A 8~20페이지.
- 금지. "견조한", "안정적인", "선방" 같은 형용사 자랑. 변동 원인 누락. 가이던스 회피.

### LP 리포트 (펀드 업데이트)
- 종결 체. `~이다` 체.
- 문체. 분기 메시지 한 줄을 머리에. 포트폴리오 별 한 단락(NAV·MOIC·IRR). 펀드 전반 요약.
- 분량. 분기 10~30페이지, 연차 50~100페이지.
- 금지. 자기 변호 일색. 비교 기준 변경 무고지. 손실 회피 표현. 약속의 인플레이션.

### 비즈니스 실사 메모 (Commercial DD)
- 종결 체. `~이다` 체. 시사점 줄은 메모 톤.
- 문체. 가설을 한 줄로 시작. 검증 방법(인터뷰·서베이·데이터). 결과 → 시사점.
- 분량. 5~20페이지. 한 가설당 2~5페이지.
- 금지. 가설 없이 자료 나열. 인용 출처 부재. 정량과 정성의 분리 누락.

### 산업·시장 분석 리포트 (Sector Note)
- 종결 체. `~이다` 체.
- 문체. 산업 변화의 인사이트 한 줄. 드라이버·승자·패자 셋. 데이터 출처와 함께.
- 분량. 5~15페이지. 차트 5~10개.
- 금지. 데이터 나열로 끝. 출처 누락. "추가 분석이 필요"로 끝맺기.

### 전문가 콜 노트 (Expert Call)
- 종결 체. `~이다` 체. 인용은 직접 인용 그대로.

## fallback 5단계 합성 알고리즘

1. **결정자의 위치를 본다.** 임원 / 애널리스트 / LP / 딜 매니저 / 클라이언트 / 재무팀장 중 누구인가.
2. **분석의 종류를 본다.** 정량(모델·숫자) / 정성(전문가 콜·인터뷰) / 종합(진단·DD) / 정기 리포트 중 무엇인가.
3. **두 후보를 고른다.** 결정자 일치 + 분석 종류 일치가 1순위.
4. **문체 합성.** 통찰 한 줄 + 데이터 출처 + 메모 톤 결론 권고를 골격으로. 9.7 영역이면 BLACK 제약 두 가지 이상.
5. **금지 합성.** 데이터 나열 + "추가 분석이 필요" + 형용사 자평 세 줄을 기본 금지로.

예. **"경쟁사 분석 메모"**. 결정자 임원. 분석 종류 정량+정성 종합. 후보 = 시장 분석 + 분석 보고서. 합성. 통찰 한 줄 + 경쟁사별 표 + 시사점·권고 메모 톤. 금지. 데이터 나열 + "추가 분석이 필요"로 끝맺기.
